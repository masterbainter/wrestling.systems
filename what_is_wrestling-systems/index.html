üìã What is Wrestling Systems?

Wrestling Systems is a comprehensive coaching platform designed to help wrestling coaches create, organize, and share their training methodologies with athletes. The platform enables structured skill tracking, progress monitoring, and parent involvement in athlete development.

Core Concept: Coaches build custom "Systems" (collections of techniques and components) which are then used to rate and track athlete progress over time.

Coach Workflow
Coaches are the content creators and evaluators. They build wrestling systems, manage athletes, and provide ongoing skill assessments.
1
Account Creation & Login
Register at /register/ with email and password, selecting "Coach" as role.
What You Get:
Access to coach dashboard with sidebar navigation featuring:
  • Dashboard: Overview of systems, athletes, and recent ratings
  • Rate Athletes: Quick access to rating workflow
  • My Athletes: Athlete roster management
  • Component Library: Browse all techniques across systems
  • Systems: Manage your wrestling systems
2
Build Your Wrestling System
Navigate to Systems ‚Üí Create System
  • Name your system (e.g., "Folkstyle Fundamentals", "Advanced Pins")
  • Select wrestling style (Folkstyle, Freestyle, Greco-Roman)
  • Add description of methodology
  • Choose visibility (Public = shareable, Private = coach-only)
System Examples:
  • Demo Takedowns: Fundamental takedown techniques
  • Demo Escapes: Bottom position escape techniques
  • Demo Pins: Pinning combinations
3
Add Components (Techniques)
Within each system, create components representing individual techniques:
  • Component name (e.g., "Single Leg Takedown", "Stand-Up Escape")
  • Detailed description of technique execution
  • Hierarchical structure (parent-child for complex moves)
  • Version tracking (updates preserve rating history)

Manage Athlete Roster
Go to My Athletes to view and manage your roster:
  • Search/Filter: Quick search by name, age, verification status
  • Quick Assign: One-click assignment to your coaching roster
  • Verification: Athletes appear after parent verification (COPPA compliant)
  • View Progress: Click athlete to see rating history across all components

5
Rate Athletes
Navigate to Rate Athletes for the main coaching workflow:
  • Select athlete from your roster
  • Choose system and component to rate
  • Assign score (1-10 scale)
  • Add optional notes/feedback (max 1000 characters)
  • Submit rating (creates timestamped entry with component version tracking)
Rating Features:
  • Historical Tracking: One rating per day (same-day updates allowed)
  • Progress Timeline: View improvement over time (6‚Üí7‚Üí8‚Üí9)
  • Version Awareness: System flags when component updated after rating
  • Bulk Rating: Rate multiple athletes on same component efficiently
6
Review Analytics
Access Dashboard for coaching insights:
  • Score Distribution: Histogram showing rating patterns
  • High Achievers: Athletes excelling in specific components
  • Struggling Components: Techniques needing more coaching attention
  • Recent Activity: Latest ratings and athlete progress updates
  • Time Filters: View data by last 7 days, 30 days, or all time
7
Share Systems
Make systems public to share with the wrestling community:
  • Toggle system visibility to "Public"
  • Other coaches can browse and clone your systems
  • Build reputation as methodology expert
  • Contribute to collective coaching knowledge base

Athlete Workflow
Athletes view their progress, track skill development, and understand their training focus areas.
1
Account Creation (Parent-Initiated)
Athletes cannot self-register. A parent/guardian must create the account:
  • Parent goes to /register/ and selects "Create Athlete Account"
  • Enters athlete details: name, email, date of birth
  • Parent receives verification email
  • After verification, athlete account is activated
  • Athlete receives login credentials via parent's email
‚ö†Ô∏è COPPA Protection: This ensures parental consent for athletes under 13 and prevents unauthorized account creation.
2
Login & Dashboard Access
Login at /login/ with credentials provided by parent.
Athlete Dashboard Features:
  • Progress Overview: Current skill levels across all components
  • Recent Ratings: Latest feedback from coaches
  • Improvement Trends: Visual graphs showing progress over time
  • Coach Assignments: View which coaches are tracking your development
3
View Ratings & Feedback
Navigate to My Ratings to see detailed assessments:
  • Filter by system, component, or coach
  • View rating timeline (track improvement from 5‚Üí7‚Üí9)
  • Read coach notes and feedback for each rating
  • Identify strengths and areas needing work
4
Study Components
Access Technique Library to review wrestling techniques:
  • Browse all components assigned by coaches
  • Read detailed descriptions of proper execution
  • Understand hierarchical technique relationships
  • Focus practice on lower-rated components
5
Track Goals
Set and monitor personal improvement goals:
  • Target specific component scores (e.g., "Reach 8/10 on Single Leg")
  • View progress toward goals over time
  • Celebrate achievements when goals are met

Parent Workflow
Parents create athlete accounts, monitor their child's progress, and stay informed about their wrestling development.
1
Account Creation & Verification
Register at /register/ selecting "Parent" as role:
  • Provide parent email and password
  • Verify email address via confirmation link
  • Complete parent profile setup
2
Create Athlete Account(s)
From parent dashboard, click Add Child Athlete:
  • Enter athlete name, email, and date of birth
  • System validates age (COPPA compliance for under 13)
  • Parent receives verification email
  • Click verification link to activate athlete account
  • Receive athlete login credentials
  • Share credentials with child
Multi-Child Management:
Parents can manage multiple athlete children from a single dashboard, viewing all their ratings and progress in one place.
3
Monitor Athlete Progress
Navigate to My Children dashboard:
  • View all registered athletes under your account
  • See summary of recent ratings and progress
  • Click individual athlete to view detailed progress
  • Review coach feedback and notes
  • Track skill development over time
4
Review Coach Assignments
See which coaches are working with your child:
  • View list of assigned coaches
  • See which systems each coach is using
  • Review rating history by coach
  • Contact coaches if questions arise
5
Support Development
Use platform insights to support athlete growth:
  • Identify components needing extra practice
  • Celebrate progress and improvements
  • Discuss goals with athlete based on ratings
  • Coordinate with coaches on development focus
